Quick Start

# 1. Clone & Install
git clone https://github.com/ab180/airflux-agent-platform.git
cd airflux-agent-platform
npm install

# 2. LLM 연결 (택 1)
claude login          # Claude Code 구독 (Anthropic)
codex login           # Codex CLI 구독 (OpenAI)
# 또는 API 키 직접 설정
export ANTHROPIC_API_KEY=sk-ant-...  # Anthropic
export OPENAI_API_KEY=sk-...         # OpenAI

# 3. 실행
npm run dev
# Server:    http://localhost:3000
# Dashboard: http://localhost:3001

claude login 또는 codex login만 하면 API 키 없이 바로 동작합니다. 대시보드 설정 페이지에서 버튼으로 CLI 로그인을 실행할 수도 있습니다.

주요 기능

1 보안

19패턴 prompt injection 차단, PII 마스킹, SSRF 방어, 출력 가드레일, timing-safe 인증

2 AI 모니터링

per-token 비용 추적, 일별 예산 한도, 실행 상태 머신, 크래시 복구

3 Slack 연동

Event API + Slash command, 스레드 세션, guardrail 적용

4 Advisor 패턴

Sonnet(저비용) executor + Opus(고품질) advisor 조합

5 Docker + PostgreSQL

docker compose up으로 PostgreSQL + 서버 즉시 실행. 채팅 히스토리 무제한.

Docker 실행

# PostgreSQL + 서버 동시 실행
docker compose up --build

# 또는 PostgreSQL만 띄우고 로컬 개발
docker compose up postgres -d
DATABASE_URL=postgres://airflux:airflux@localhost:5432/airflux npm run dev

API

# 에이전트에 질문
curl -X POST http://localhost:3000/api/query \
  -H "Content-Type: application/json" \
  -d '{"query": "DAU가 뭐야?"}'

# 디버그 모드
curl -X POST http://localhost:3000/api/query \
  -H "Content-Type: application/json" \
  -d '{"query": "debug:리텐션 설명해줘"}'
